# Спецификация дизайна sOn Messenger — стиль MAX/Telegram Desktop

## Референс: VK Мессенджер MAX (desktop macOS app)

Этот документ описывает ТОЧНЫЙ дизайн для воспроизведения в webApp.
Все размеры, цвета, отступы взяты из скриншотов MAX.

---

## 1. Общий Layout (3 колонки)

```
┌──────────┬──────────────────┬─────────────────────────────┐
│ Sidebar  │   List Panel     │      Content Panel          │
│  68px    │    360px         │      flex-1                 │
│          │                  │                             │
│  Все     │  Заголовок + [+] │  Шапка чата (аватар, имя)   │
│  Каналы  │  Поиск           │  Сообщения                  │
│  Контакты│  Список чатов    │  Поле ввода                 │
│  Звонки  │                  │                             │
│          │                  │                             │
│ ──────── │                  │                             │
│ Настройки│                  │                             │
└──────────┴──────────────────┴─────────────────────────────┘
```

### Sidebar (68px)
- **Фон**: `#16161a` (очень тёмный, почти чёрный)
- **Ширина**: 68px фиксированная
- **Разделитель справа**: `1px solid rgba(255,255,255,0.06)`
- **Иконки**: 22px, Lucide icons, вертикально по центру
- **Текст под иконкой**: 10px, одна строка
- **Активный таб**: фон `rgba(255,255,255,0.08)`, текст белый, rounded-xl
- **Неактивный**: цвет `rgba(255,255,255,0.45)`
- **Badge**: круг 18px, фон `#5B5FC7` (фиолетовый), текст 10px bold белый
- **Верхние табы**: Все (MessageSquare), Каналы (Hash), Контакты (Users), Звонки (Phone)
- **Нижний таб**: Настройки (Settings) — прижат к низу
- **Отступы**: py-3 px-1, gap-1 между табами

### List Panel (360px)
- **Фон**: `#1e1e2e` (тёмно-синий)
- **Ширина**: 360px фиксированная
- **Разделитель справа**: `1px solid rgba(255,255,255,0.06)`
- **Заголовок**: "Чаты" 22px bold белый, справа кнопка [+] (синий круг 36px)
- **Поиск**: input с иконкой лупы, фон `#282840`, rounded, placeholder "Найти"
- **Scroll**: тонкий скроллбар 6px, thumb rgba(255,255,255,0.15)

### Content Panel (flex-1)
- **Фон**: `#141420` (самый тёмный)
- **Фоновый паттерн**: тёмный узор (звёзды/космос) с opacity 0.03 — опционально
- **Пустое состояние**: иконка + "Выберите чат" по центру, text-white/30

## 2. Список чатов (Chat List Items)

Каждый элемент:
- **Высота**: ~72px
- **Padding**: px-4 py-3
- **Hover**: `rgba(255,255,255,0.04)`
- **Активный**: `rgba(91,95,199,0.15)` (фиолетовый tint)
- **Аватар**: 48px круглый, слева
- **Имя**: 15px semibold белый, truncate, справа — время (13px, rgba(255,255,255,0.35))
- **Превью**: 14px, rgba(255,255,255,0.45), truncate одна строка, макс 2 строки
- **Unread badge**: справа от превью, круг 20px, фон `#5B5FC7`, текст 12px bold белый
- **Pinned icon**: маленькая булавка справа от имени, rgba(255,255,255,0.3)
- **Muted icon**: перечёркнутый колокольчик, rgba(255,255,255,0.2)
- **Разделитель**: нет видимого бордера между элементами (чистый вид)

### Кнопка [+] dropdown
При нажатии на "+" — popup с пунктами:
- Создать группу
- Создать приватный канал
- Создать групповой звонок
- Найти по номеру
- Пригласить по ссылке

Popup: фон `#282840`, rounded-xl, shadow, каждый пункт py-3 px-4, hover bg rgba(255,255,255,0.06)

---

## 3. Шапка чата (Chat Header)

- **Фон**: `#1e1e2e` (такой же как list panel)
- **Высота**: ~60px
- **Разделитель снизу**: `1px solid rgba(255,255,255,0.06)`
- **Слева**: Аватар 40px + Имя (17px semibold) + Подзаголовок (13px, "1 271 232 подписчика" или "был(а) в сети")
- **Справа**: кнопка поиска (лупа, 20px)
- **Для каналов**: имя канала + число подписчиков

---

## 4. Сообщения (Message Bubbles)

### Стиль пузырей (тема "Космос")
- **Исходящие**: фон `#5B5FC7` (фиолетовый), скруглённые углы 18px
- **Входящие**: фон `#282840` (тёмно-серый), скруглённые углы 18px
- **Текст**: 15px, белый, line-height 1.4
- **Время**: 12px, rgba(255,255,255,0.5), справа внизу пузыря
- **Галочки доставки**: две галочки (✓✓) рядом с временем, rgba(255,255,255,0.5)
- **Максимальная ширина**: 65%
- **Отступ между пузырями группы**: 2px
- **Отступ между группами разных авторов**: 8px

### Реакции
- Под пузырём, слитно
- Фон: `rgba(255,255,255,0.08)`, rounded-full
- Текст: emoji + число, 12px
- Отступ от пузыря: -4px (перекрытие)

### Дата-разделитель
- По центру, фон `rgba(0,0,0,0.3)`, rounded-full, px-3 py-1
- Текст: 12px, rgba(255,255,255,0.6)
- "28 марта", "Вчера", "Сегодня"

### Forwarded messages
- Заголовок: "Переслано:" 12px, rgba(255,255,255,0.5)
- Имя отправителя: 13px bold, цвет акцента
- Разделитель: вертикальная линия 3px слева от цитаты, цвет акцента

### Файлы в пузыре
- Иконка типа файла (DOCX синий, PDF красный, LOG зелёный)
- Имя файла: 14px bold
- "Скачать XX КБ": 12px, rgba(255,255,255,0.5)
- Стрелка скачивания рядом с иконкой

### URL Preview
- Внутри пузыря: ссылка кликабельная (синяя)
- Под ссылкой: карточка с image + title + description + favicon
- Фон карточки: rgba(255,255,255,0.06), rounded

### Каналы — посты
- Имя канала вверху поста (16px bold)
- "Переслано: Малек Дудаков" — 13px
- Жирный текст: **bold** рендеринг
- Ссылки: синие, underline
- Счётчик просмотров: "👁 291 · 13:30" внизу
- Реакции: горизонтально под постом
- "Прокомментировать →": кнопка внизу, фон rgba(255,255,255,0.08)

## 5. Поле ввода (Input Bar)

- **Фон панели**: `#1e1e2e`
- **Разделитель сверху**: `1px solid rgba(255,255,255,0.06)`
- **Слева**: кнопка скрепки (вложения), 20px, rgba(255,255,255,0.45)
- **Центр**: textarea, фон `#282840`, rounded-2xl, placeholder "Сообщение", 15px
- **Справа**: смайлик + микрофон (если текст пустой) или кнопка отправки (если есть текст)
- **Popup вложений**: "Фото или видео", "Файл" — dropdown вверх от скрепки

## 6. Настройки (Settings)

### Левая панель (внутри List Panel 360px)
- **Заголовок**: "Настройки" 22px bold
- **Профиль**: Аватар 60px + Имя (17px bold) + Телефон (14px, rgba(255,255,255,0.5)) + chevron
- **"Пригласить друзей"**: иконка ссылки + текст синий
- **Разделители между секциями**: тонкая линия rgba(255,255,255,0.06)
- **Пункты меню**: иконка (20px, rgba(255,255,255,0.5)) + текст (15px) + chevron справа
  - Безопасность (Shield)
  - Устройства (Monitor)
  - Папки (Folder)
  - —— разделитель ——
  - Оформление (Palette)
  - Язык приложения (Globe)
  - Сочетания клавиш (Keyboard)
  - —— разделитель ——
  - Помощь (HelpCircle)
  - О приложении (Info)
- **Активный пункт**: фон rgba(255,255,255,0.06), текст белый

### Правая панель (Content Panel)
Зависит от выбранного раздела:

#### Профиль (редактирование)
- Аватар 80px с кнопкой камеры (синий кружок)
- Имя: input, 16px
- Фамилия: input
- О себе: textarea
- "Выйти из профиля": кнопка + "..."

#### Безопасность
- Пароль для входа: toggle/chevron
- Семейная защита: toggle
- Безопасный режим: toggle
- Позвонить / Найти по номеру / Пригласить в чат / Показывать контент: "могут контакты" + замок
- Видеть статус «в сети»: "могут контакты"
- Чёрный список: chevron

#### Устройства
- Заголовок + иконка монитора
- Текст "Входите на новых устройствах и управляйте сессиями"
- Список сессий: Имя устройства (bold) + детали (OS, IP) + статус ("В сети" зелёный / время)
- "Завершить все сессии, кроме текущей": красная кнопка

#### Папки
- "Создать папку" кнопка (синяя)
- Список папок: Все, Сферум, Новые, Каналы — с drag handle и "..."
- Шаблоны: "Личные" + [+]

#### Оформление
- Тема: Системная / Светлая / Тёмная (tab switcher)
- Тема оформления: превью чата с пузырьками
- Выбор темы: карточки (Космос, Уверенная, Изящная, Летний лес) с цветовыми квадратами

#### Язык
- Русский + флаг + radio (активный)
- English + флаг + radio
- "Язык изменится только на этом устройстве"

#### Сочетания клавиш
- Секции: "Отправлять сообщение", "Поиск и навигация", "Работа с сообщениями", "Форматирование текста"
- Каждый пункт: описание + shortcut (Cmd+F, Cmd+K, Esc, Cmd+B, Cmd+I, Cmd+U)

#### Помощь
- "Ответы на вопросы" заголовок
- Карточки: Начало работы, Профиль, Безопасность, Госуслуги, Переписка, Управление групповым чатом

## 11. Анимации

- Переход между табами sidebar: fade 150ms
- Открытие чата: slide-in-right 200ms
- Hover на элементах: background transition 150ms
- Badge появление: scale spring 200ms
- Popup появление: fade + scale 150ms
